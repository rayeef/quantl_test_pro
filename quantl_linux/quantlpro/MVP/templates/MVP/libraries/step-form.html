Step Form (Wizard)

Creates a multi step form to get more leads and increase engagement.

How to use?

Wrap any block in a parent element with an ID or class and add the same ID in the JS init function of the hs.validation.js plugin.

Copy-paste the following <script> near the end of your pages under JS Nova to enable it.

            
              <script src="../../assets/js/components/hs.jquery-validation.js"></script>
              <script src="../../assets/js/components/hs.step-form.js"></script>

Copy-paste the init function under JS Plugins Init., before the closing </body> tag, to enable it.

            
              <script>
                $(document).on('ready', function () {
                  // initialization of validation
                  $.HSCore.components.HSValidation.init('#stepForm');

                  // initialization of step form
                  $.HSCore.components.HSStepForm.init('#stepForm', {
                    validate: function() {
                      return $('#stepForm').valid();
                    }
                  });
                });
              </script>

JavaScript behavior

Config 

            
              dotTemplate: '',
              activeClass: 'active',
              currentStep: 0,
              validate: function () {}
Parameter Description
dotTemplate single dot pattern.
activeClass class that will be given to the block with the active step.
currentStep determines the initial form to fill out.
validate performs all the functions inside, each time while validating the whole form.

Methods

Options can be passed via data attributes or JavaScript. For data attributes, append the option name to data-, as in data-current-step="".

Attribute Description
data-current-step determines the initial form to fill out.
data-steps-selector The block ID that contains the blocks with steps.
data-counter-selector The block ID that contains a step counter.
data-dots-selector The block ID into which dots are generated.
data-next-selector ID of the element, when clicked, which will go to the next step.
data-submit-selector ID of the element, when clicking on which the form will be sent.
